本项目所需要使用到的python依赖库，除了基本库以外还需要:
numpy
pandas
re
scikit-learn

强烈推荐安装使用anaconda。它除了囊括所有我们所需要的库以外（可能没有re），还提供了Jupyter Notebook，一款可以进行数据分析，即时检验以及进行汇报分享交流的十分有用的数据科学工具。我们提供的demo也基于此。

可以通过以下任意方式获取Anaconda：
官方安装下载地址：https://www.anaconda.com/download/
清华大学开源软件站：https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/


我们提供了一个scripts的包，里面包括了preprocessing.py和naive_bayes.py两个脚本文件。分别对应了数据清洗和朴素贝叶斯分类的需求。
具体的使用办法，如何调用，我们在demo.ipynb文件中详细介绍了。该文件最好在Jupyter Notebook中打开。如果使用其他ide打开，将会出现markdown代码，在阅读上会造成一些困难。
